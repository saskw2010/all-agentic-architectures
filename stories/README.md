# Skyverse Stories

Interactive narrative subapplications for the Sky365 Agentic Knowledge Platform.

## Current stories

- `turing-trail/` — **The Turing Trail: From Rules to Agents**

## Publishing model

The GitHub Pages workflow copies this directory to `_site/stories/`, preserving every story as an independent route.

## Story principles

1. Start with the scientific or historical question, not product marketing.
2. Prefer original and official sources.
3. Separate verified facts from interpretation.
4. Expose the limits and trade-offs of each technology.
5. Connect the story to SKY365 architecture only after the underlying ideas are clear.
6. Keep every story responsive, accessible, dependency-light, and compatible with reduced-motion preferences.

## Adding another story

Create a new folder under `stories/`, add its `index.html`, and register it in `stories/index.html`. The deployment workflow publishes the folder automatically.
