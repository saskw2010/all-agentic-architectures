LM Studio Troubleshooting

تشخيص سريع لمشاكل التحميل، paging، البطء، جودة العربية، وفجوة الـDemo عن الإنتاج.

Model Does Not Load

  • قلّل context.
  • استخدم quantization أصغر.
  • قلّل GPU offload.
  • أغلق التطبيقات الثقيلة.
  • راجع مساحة القرص والـpage file.

System Unresponsive

غالبًا يوجد paging كثيف. استخدم نموذجًا أصغر، context أقل، أو concurrent workloads أقل.

Slow Output

  • قلّل context.
  • اختبر طبقات GPU أقل.
  • انقل النماذج إلى NVMe.
  • لا تحمل عدة نماذج معًا.

Weak Arabic

اختبر العربية منفصلة. جودة coding لا تضمن جودة لغة الأعمال العربية.

Good Demo, Bad Production

Prompt واحد ناجح لا يثبت شيئًا. استخدم benchmark ثابتًا بعد كل تغيير في model أو quantization أو context أو runtime.

Unsafe Agent Behavior

  • DraftOnly=true
  • Validate structured output
  • Block direct DB writes
  • Route through Shared Action Core
  • Require policy-based approval
  • Persist trace and decision evidence
